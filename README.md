# Pré requisitos

Download and Install Node & NPM: https://nodejs.org/en/

Download and install GIT: https://git-scm.com/downloads

# Instalação

```
npm install
```

# Iniciar servidor de desenvolvimento

```
npm run dev
```

# Compilar versão de produção

```
npm run build
```

# Iniciar servidor de produção

```
npm run start
```
